# rustfetch
Just another system information tool written in Rust.  

Features:  
- Minimal
- Dependency-free
- Built with rust!

## Config
rustfetch configuration is located at ~/.config/rustfetch

### Sample config
```sh
User: `user`
Uptime: `uptime`
Shell: `shell`
```  
rustfetch will automatically replace user, uptime, and shell with their appropriate values.
